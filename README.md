# Vehicle Dynamics Calculator
A Python project exploring automotive engineering concepts through practical vehicle dynamics calculations.

## Features
✅ Braking Distance Calculator
✅ Reaction Distance Calculator
✅ Total Stopping Distance Calculator
✅ Power to Weight Ratio

## Planned Features
- Gear Ratio Calculator
- Engine RPM Calculator
- Aerodynamic Drag Calculator

## How to Run

```bash
python3 braking.py
```
